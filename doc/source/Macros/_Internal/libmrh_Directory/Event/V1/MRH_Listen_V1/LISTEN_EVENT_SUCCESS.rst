LISTEN_EVENT_SUCCESS
====================
The LISTEN_EVENT_SUCCESS macro defines the value indicating 
success for a listen event.

Header
------
The macro definition can be found in the following file:

.. code-block:: c

    libmrh/Event/V1/MRH_Listen_V1.c


Definition
----------
The defined value in the source is as follows:

.. code-block:: c

    #define LISTEN_EVENT_SUCCESS 0
