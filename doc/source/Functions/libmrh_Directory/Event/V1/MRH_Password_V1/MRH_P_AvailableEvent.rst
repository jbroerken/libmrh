MRH_P_AvailableEvent
====================
The MRH_P_AvailableEvent function is used to create a 
password service available event.

Header
------
The function definition can be found in the following file:

.. code-block:: c

    #include <libmrh/Event/V1/MRH_Password_V1.h>


Syntax
------
.. code-block:: c

    MRH_Event* MRH_P_AvailableEvent(void);


Parameters
----------
None.

Return Value
------------
A password service available event on success, 
NULL on failure.

-------
None.
