MRH_N_AvailableEvent
====================
The MRH_N_AvailableEvent function is used to create a 
notification service available event.

Header
------
The function definition can be found in the following file:

.. code-block:: c

    #include <libmrh/Event/V1/MRH_Notification_V1.h>


Syntax
------
.. code-block:: c

    MRH_Event* MRH_N_AvailableEvent(void);


Parameters
----------
None.

Return Value
------------
A notification service available event on success, 
NULL on failure.

-------
None.
