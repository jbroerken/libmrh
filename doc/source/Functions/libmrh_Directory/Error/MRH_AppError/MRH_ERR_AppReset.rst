MRH_ERR_AppReset
================
The MRH_ERR_AppReset function is used to reset the current 
library error value.

Header
------
The function definition can be found in the following file:

.. code-block:: c

    #include <libmrh/Error/MRH_AppError.h>


Syntax
------
.. code-block:: c

    void MRH_ERR_AppReset(void);
