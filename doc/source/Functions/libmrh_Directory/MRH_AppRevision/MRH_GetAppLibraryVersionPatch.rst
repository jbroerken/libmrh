MRH_GetAppLibraryVersionPatch
=============================
The MRH_GetAppLibraryVersionPatch function is used to obtain 
the library patch version.

Header
------
The function definition can be found in the following file:

.. code-block:: c

    #include <libmrh/MRH_AppRevision.h>


Syntax
------
.. code-block:: c

    int MRH_GetAppLibraryVersionPatch(void);


Parameters
----------
None.

Return Value
------------
The library patch version defined by 
:doc:`../../../Macros/libmrh_Directory/MRH_AppRevision/MRH_APP_LIB_VERSION_PATCH`.

Remarks
-------
None.
